## 2. Background and Literature Review

### 2.1 Vector Indexing Fundamentals

Vector similarity search aims to efficiently retrieve nearest neighbors from large datasets of high-dimensional vectors. Exact search is often computationally prohibitive, motivating approximate nearest neighbor (ANN) methods that trade some accuracy for speed and scalability.

- **HNSW** constructs a multi-layered graph where each node represents a data point connected via navigable small-world links. - **IVF-PQ** combines two complementary techniques: an inverted file (IVF) partitions the dataset into clusters, limiting the search scope to relevant partitions; product quantization (PQ) compresses vectors into compact codes representing centroid indices, enabling fast approximate distance computations with reduced memory footprint [1][3][4].
